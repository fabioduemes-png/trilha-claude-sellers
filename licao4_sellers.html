Os três modos do app desktop

O app desktop do Claude oferece três formas de trabalhar. Cada modo foi projetado para um tipo de tarefa. Clique em cada um para conhecer os detalhes.

💬
Chat
Perguntas rápidas, rascunhos iterativos e aprendizado por diálogo
🗂️
Cowork
Pesquisa aprofundada, análise de múltiplas fontes e entregas complexas
⌨️
Code
Desenvolvimento de software: escrever, testar e implantar código
💬 Chat Trocas rápidas
⚡
Acesso rápido Toque duas vezes em Option (Mac) para abrir o Claude sobre qualquer janela. Sem sair do que você estava fazendo.
📸
Capturas de tela Capture a tela ou compartilhe uma janela para o Claude ver exatamente o que você está olhando.
🎙️
Ditado por voz Fale sobre um problema em vez de digitar. Útil quando pensar em voz alta é mais rápido que escrever.
🔌
Conectores desktop Conecte ferramentas e serviços locais para que o Claude trabalhe com outros apps na sua máquina.
Use o Chat quando...
→Você está olhando para um painel e quer entender as métricas sem sair da janela
→Está entre reuniões e quer estruturar uma apresentação por voz
→Precisa de uma resposta rápida sem abrir uma nova aba
🗂️ Cowork Trabalho complexo
📁
Acesso a pastas Dê ao Claude acesso a uma pasta no seu computador. Ele lê, identifica o que é relevante e salva o resultado no mesmo lugar.
🕐
Tarefas agendadas Configure rotinas automáticas: briefing diário, resumo semanal, triagem matinal. O Claude executa enquanto o app estiver aberto.
🌐
Uso do navegador Com Claude no Chrome, ele navega em sites, interage com páginas e traz dados diretamente para a tarefa — sem você abrir uma aba.
🧩
Plugins Adicione capacidades extras: dados financeiros em tempo real, base de conhecimento interna, frameworks específicos de conformidade.
Use o Cowork quando...
→Precisa cruzar informações de e-mails, Slack, reuniões e documentos de uma vez
→Está mapeando concorrentes ou avaliando ferramentas com múltiplas fontes
→Faz o mesmo trabalho toda manhã e quer automatizar com tarefa agendada
⌨️ Code Desenvolvimento
👁️
Diffs visuais Veja exatamente o que o Claude alterou no código antes de aceitar qualquer mudança.
🖥️
Terminal integrado Comandos executados em tempo real, visíveis no painel — você acompanha cada passo.
🔀
Três modos de controle Ask: Claude propõe, você aprova. Code: aplica automaticamente, pede confirmação no terminal. Plan: descreve toda a abordagem antes de tocar no código.
☁️
Local ou remoto Trabalhe direto na sua máquina ou conecte um repositório GitHub. No remoto, a sessão continua mesmo com o app fechado.
Use o Code quando...
→Quer construir funcionalidades descrevendo em linguagem natural
→Precisa depurar erros colando a mensagem de erro e deixando o Claude investigar
→Está navegando em uma base de código desconhecida
💡 Cowork e Code usam o mesmo motor. Ambos rodam o Claude Code por baixo — localmente na sua máquina, capazes de trabalhar de forma independente e sustentar tarefas longas.

💡 O Chat é igual ao claude.ai — com vantagens extras por rodar nativamente no computador. O Cowork e o Code são para tarefas que precisam de autonomia e mais alcance.
